# 🧠 Diabetes Prediction ML Web App

## 🚀 Overview

This project is a full-stack machine learning web application that predicts whether a person is likely to have diabetes based on medical input features.

The application integrates a trained classification model built using **ML.NET** with a modern **Blazor UI**, allowing users to input data and receive real-time predictions.

---

## 🏗️ Tech Stack

* 💻 Frontend: Blazor (MudBlazor UI)
* ⚙️ Backend: .NET / C#
* 🤖 Machine Learning: ML.NET
* 📊 Data Processing: CSV dataset

---

## 📂 Project Structure

* `/Components` → UI pages and layout
* `/mlstuff` → ML model and prediction logic
* `/wwwroot` → static assets
* `Program.cs` → app entry point

---

## ✨ Features

* Interactive UI for user input
* Real-time diabetes prediction
* Integrated ML model inside web app
* Clean and modular architecture

---

## ▶️ How to Run

1. Clone the repository
2. Open in Visual Studio
3. Build and run the project
4. Enter input values and get predictions
