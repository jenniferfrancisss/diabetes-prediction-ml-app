# 🧠 Diabetes Prediction ML Web App

## 🚀 Live Demo

🔗 https://diabetes-prediction-ml-app-3wfm.onrender.com

---

## 📌 Overview

This project is a **full-stack machine learning web application** that predicts whether a person is likely to have diabetes based on medical input features.

It combines a trained classification model built using **ML.NET** with a modern, interactive **Blazor (MudBlazor)** user interface, enabling real-time predictions directly in the browser.

---

## 🏗️ Tech Stack

* 💻 **Frontend:** Blazor (MudBlazor UI)
* ⚙️ **Backend:** .NET / C#
* 🤖 **Machine Learning:** ML.NET
* 📊 **Data Processing:** CSV Dataset
* ☁️ **Deployment:** Docker + Render

---

## ✨ Features

* Interactive sliders for medical input parameters
* Real-time diabetes prediction
* Seamless integration of ML model into web app
* Clean, modular, and scalable architecture
* Fully deployed cloud application

---

## 📂 Project Structure

* `/Components` → UI pages and layouts
* `/mlstuff` → ML model and prediction logic
* `/wwwroot` → Static assets
* `Program.cs` → Application entry point

---

## ▶️ How to Run Locally

1. Clone the repository
2. Open in Visual Studio
3. Restore dependencies
4. Build and run the project
5. Enter input values and click **Predict**
